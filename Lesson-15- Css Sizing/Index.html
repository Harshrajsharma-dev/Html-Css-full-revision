CSS Sizing Units

1. Absolute Unit: px

Pixels (px) represent fixed sizes. Best used for elements where consistent dimensions are needed—like borders, icons, or fixed-width layouts.

2. Relative Units

em

Relative to the font-size of the parent element. For example, if the parent has font-size: 16px, then 2em equals 32px.

rem

Relative to the root <html> font size. Ensures consistent sizing across the entire document and is great for global responsiveness.

3. Viewport Units

vh & vw

  • 1vh = 1% of the viewport height
  • 1vw = 1% of the viewport width

Best for responsive full-screen sections, hero banners, and adaptive typography.

4. Percentages %

Relative to the size of the parent container. Commonly used for width, padding, and margins in fluid layouts.

5. When to Use Each Unit

Unit Use Case
px Precise control (e.g., icons, borders)
em Nesting & scalable components
rem Global scaling & consistent layout
vh / vw Full-screen sections & responsive spacing
% Flexible layouts relative to parent

6. Additional Units

  • vmin – based on the smaller of viewport height/width
  • vmax – based on the larger of viewport height/width
  • ch, ex – font-based units (less common)

✅ Summary

Absolute: px → fixed size
Relative: em, rem → font-based
Viewport: vh, vw → screen-based
Percentage: % → relative to parent
