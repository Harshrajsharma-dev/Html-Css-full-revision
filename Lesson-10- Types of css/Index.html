Types of CSS

  • Inline CSS = It is used inside a tag. 
    <h1 style="color: blue;">Hello World</h1>
  • Internal CSS = It is used within a style tag. 
    <style>
    h1 {
        color: green;
    }
</style>
  • External CSS = It is used in a separate file. 
    <link rel="stylesheet" href="styles.css">
