CSS Font, Text & Color Properties

1. Font Basics

Use font-family to specify the typeface, with fallbacks:

font-family: "Roboto", Arial, sans-serif;

The browser picks the first available font from the list.

2. Font Weight & Style

font-weight controls thickness: normal (400), bold (700), or numeric (100â€“900).

font-style offers options like italic, oblique, or normal.

3. Font Size

Use relative units like em or rem for scalable text.

Absolute units like px set fixed size.

4. Line Height & Letter Spacing

line-height improves readability by controlling vertical spacing.

letter-spacing adjusts spacing between individual letters.

5. Font Color

Define text color with color using hex (#333), rgb (rgb(51,51,51)), or named colors (black).

6. Including External Fonts

Use Google Fonts via a <link> in <head>:

<link href="https://fonts.googleapis.com/css?family=Roboto:wght@400;700" rel="stylesheet">

Then apply it with CSS:

body {
  font-family: "Roboto", sans-serif;
}

7. Text Transformations

text-transform controls capitalization: uppercase, lowercase, capitalize

8. Decorative Text Properties

text-decoration adds styles like underline, line-through

text-shadow adds depth and emphasis.
