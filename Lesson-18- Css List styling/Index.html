2. Changing List Style Type

Use list-style-type to change bullet or number type.

ul {
  list-style-type: square;
}
ol {
  list-style-type: upper-roman;
}

Example:

  • Dog
  • Cat

  2. Beta

3. Removing List Markers

To remove bullets or numbers:

ul {
  list-style-type: none;
}

Example:

  • No bullet 1
  • No bullet 2

4. Using Image as Marker

Use list-style-image with a path to your image.

ul {
  list-style-image: url('check.png');
}

Note: Replace 'check.png' with your actual image path.

5. Marker Position

Use list-style-position to place the bullet inside or outside the content box.

ul {
  list-style-position: inside;
}

Example:

  • Inside Position
  • Aligned with text

6. Shorthand Property

You can use the list-style shorthand to combine type, position, and image.

ul {
  list-style: square inside url('icon.png');
}

7. Quick Tips

  • Always use semantic list tags: <ul>, <ol>, <dl>.
  • Use CSS for styling lists instead of adding unnecessary HTML.
  • Ensure your list markers and icons render consistently across browsers.
