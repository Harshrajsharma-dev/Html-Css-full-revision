⚡ CSS Transition Property

transition-property: Defines which CSS property to animate (e.g., transform, background-color).
transition-duration: How long the animation lasts (e.g., 0.5s, 200ms).
transition-timing-function: Defines the animation's speed curve—common values are ease, linear, ease-in-out.
transition-delay: Delay before animation starts (0s by default).
Shorthand syntax: transition: <property> <duration> <timing-function> <delay>;
