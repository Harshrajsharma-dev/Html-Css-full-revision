CSS position Property

position: static (default)
Static elements stay in normal document flow. `top/left` have no effect.
position: relative; top:10px; left:20px;
Relative moves the box from its normal position; still occupies original space.
position: absolute;
Absolutely positioned relative to the nearest ancestor with `position`. Doesn't affect other elements' layout.
position: sticky; top: 0;
Scroll down within this wrapper and the heading will stick to the top.
Scroll content goes here...
position: fixed; stays here on the viewport even when page scrolls.
