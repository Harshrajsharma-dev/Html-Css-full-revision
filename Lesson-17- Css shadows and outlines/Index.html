CSS Shadows and Outlines

1. Box Shadow (box-shadow)

Used to add drop shadows to elements. Syntax:

box-shadow: h-offset v-offset blur-radius spread-radius color;

Example:

box-shadow: 4px 4px 10px rgba(0,0,0,0.3);
Box Shadow

2. Inset Shadow

Use inset to create inner shadows:

box-shadow: inset 2px 2px 5px rgba(0,0,0,0.5);
Inset Shadow

3. Multiple Shadows

You can apply more than one shadow by separating them with commas:

box-shadow: 
  2px 2px 5px rgba(0,0,0,0.3),
  inset 0 0 8px rgba(255,255,255,0.5);
Multiple Shadows

4. Outline vs Border

  • Border: Part of box model, affects layout and size.
  • Outline: Drawn outside element’s edge, doesn't affect layout.
outline: 3px dashed blue;
Outlined Box

5. Outline Offset

Use outline-offset to move the outline away from the element:

outline-offset: 4px;
Offset Outline

6. Use Cases

  • box-shadow: Adds visual depth (e.g., cards, buttons)
  • inset: Inner effects for pressed UI
  • outline: Useful for focus states (accessibility)
  • outline-offset: Adds space between element and outline

✅ Summary

box-shadow = shadow effects
inset = inner shadow
outline = non-intrusive focus border
outline-offset = spacing from element
