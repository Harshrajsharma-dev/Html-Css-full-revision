Box model

The CSS box model is a fundamental concept in web design and layout. It describes the rectangular boxes generated for elements in the document tree and consists of margins, borders, padding, and the actual content area.

Border

The border is the area around the padding (if any) and the content. It is the outermost layer of the box model and can be styled using CSS properties such as border-width, border-style, and border-color.

Padding

Padding is the space between the content and the border. It creates an inner buffer around the content, ensuring that the text or other elements do not touch the border directly. Padding can be controlled using the CSS properties padding-top, padding-right, padding-bottom, and padding-left.

Margin

Margin is the space outside the border. It creates a buffer between the element and its surrounding elements. Margins can be controlled using the CSS properties margin-top, margin-right, margin-bottom, and margin-left.
