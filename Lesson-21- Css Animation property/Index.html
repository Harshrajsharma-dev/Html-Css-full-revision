@keyframes: Defines animation stages using percentages or keywords like from/to.
animation-name: Connects the element to its @keyframes rules.
animation-duration: Specifies how long one cycle lasts (e.g., 2s).
animation-timing-function: Controls pacing (e.g., ease-in, linear, ease-in-out).
animation-delay: Adds pause before starting (0s default).
animation-iteration-count: Defines how many times it repeats (infinite for continuous loop).
animation-direction: Sets whether it alternates (normal, reverse, alternate).
animation-fill-mode: Determines styles before/after animation (forwards, backwards).
Shorthand: animation: name duration timing-function delay iteration-direction fill-mode;
